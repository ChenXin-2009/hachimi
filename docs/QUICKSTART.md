# 快速入门指南

## 安装依赖

1. 确保已安装 Python 3.8 或更高版本

2. 安装依赖包：
```bash
pip install -r requirements.txt
```

**注意**：首次运行时，Demucs 会自动下载模型文件（约 2GB），请确保网络连接正常。

## 运行应用

```bash
python main.py
```

## 基本使用流程

### 1. 导入音频文件

- 点击菜单栏 **文件 → 打开音频** 或工具栏的 **打开音频** 按钮
- 选择支持的音频文件（MP3, WAV, FLAC, OGG, M4A）
- 等待音频分离完成（进度条会显示进度）

### 2. 查看分离结果

分离完成后，会自动显示多个音轨的波形：
- **蓝色**：人声（Vocals）
- **红色**：鼓（Drums）
- **绿色**：贝斯（Bass）
- **橙色**：其他乐器（Other）

### 3. 播放音频

- 点击工具栏的 **播放** 按钮开始播放
- 点击 **暂停** 暂停播放
- 点击 **停止** 停止播放并重置到开头
- 拖动播放位置滑块可以跳转到指定位置

### 4. 导出音频

- 点击菜单栏 **文件 → 导出音频**
- 选择导出格式（WAV, FLAC, MP3）
- 选择保存位置
- 等待导出完成

### 5. 保存项目

- 点击菜单栏 **文件 → 保存项目**
- 选择保存位置
- 项目文件会保存所有音轨和参数设置

### 6. 加载项目

- 点击菜单栏 **文件 → 加载项目**
- 选择之前保存的项目文件（.json）
- 项目会恢复所有音轨和设置

## 键盘快捷键

- **Ctrl+O**：打开音频文件
- **Ctrl+S**：保存项目
- **Ctrl+E**：导出音频
- **Ctrl+Z**：撤销
- **Ctrl+Y**：重做
- **空格键**：播放/暂停（即将支持）

## 系统要求

### 最低配置
- CPU：双核处理器
- 内存：4GB RAM
- 操作系统：Windows 10 及以上

### 推荐配置
- CPU：四核处理器
- 内存：8GB RAM
- GPU：NVIDIA GPU（支持 CUDA）
- 操作系统：Windows 10 及以上

**注意**：使用 GPU 可以显著加快音频分离速度（约 5-10 倍）。

## 性能参考

处理一首 3 分钟的歌曲：
- **使用 GPU**：约 2 分钟
- **仅使用 CPU**：约 10 分钟

## 常见问题

### Q: 首次运行很慢？
A: 首次运行时需要下载 Demucs 模型文件（约 2GB），请耐心等待。模型会缓存到本地，后续运行会快很多。

### Q: 分离质量不理想？
A: 音频分离质量取决于原始音频的复杂度。对于混音复杂的音乐，分离效果可能不完美。可以尝试使用不同的模型版本。

### Q: 内存不足错误？
A: 处理长音频文件（>10 分钟）可能需要大量内存。建议：
- 关闭其他应用程序
- 处理较短的音频片段
- 增加系统内存

### Q: GPU 未被使用？
A: 确保已安装 CUDA 和对应版本的 PyTorch。应用会自动检测并使用可用的 GPU。
