# 音频转MIDI快速开始

## 5分钟上手指南

### 第1步：安装依赖（首次使用）

```bash
pip install basic-pitch
```

### 第2步：运行程序

```bash
python main.py
```

### 第3步：打开并分离音频

1. 点击 **📁 打开音频**
2. 选择你的音频文件
3. 选择模型（推荐 **htdemucs_6s**）
4. 等待分离完成

### 第4步：转换为MIDI

1. 按 **Ctrl+M** 或点击 **工具 → 音频转MIDI**
2. 选择要转换的音轨（如"vocals"人声）
3. 点击 **开始转换**
4. 保存MIDI文件

## 🎯 解决分段问题

如果"一个词被拆成多个片段"，调整这些参数：

```
音符起始阈值: 0.35 ⬇️（降低）
帧阈值: 0.20 ⬇️（降低）
最小音符长度: 200 ms ⬆️（增加）
```

## ❓ 常见问题

**Q: 音符太多？**  
A: 提高音符起始阈值到 0.6-0.7

**Q: 音符太少？**  
A: 降低音符起始阈值到 0.3-0.4

**Q: 还是有分段？**  
A: 降低两个阈值，增加最小音符长度
