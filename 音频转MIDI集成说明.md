## 🚀 使用方法

### 1. 安装依赖

```bash
pip install basic-pitch
```

### 2. 运行程序

```bash
python main.py
```

### 3. 使用流程

1. **打开音频** → 选择音频文件
2. **分离音频** → 选择模型（建议 htdemucs_6s）
3. **转换MIDI** → 点击"工具 → 音频转MIDI"（或按 Ctrl+M）
4. **选择音轨** → 选择要转换的音轨（如人声、钢琴等）
5. **调整参数** → 根据需要调整转换参数
6. **开始转换** → 点击"开始转换"并保存MIDI文件

## 🎛️ 参数调整建议

### 解决分段问题的参数设置

如果遇到"一个词被拆成多个片段"的问题：

```
推荐设置：
- 音符起始阈值: 0.35-0.45（降低）
- 帧阈值: 0.20-0.25（降低）
- 最小音符长度: 150-200 ms（增加）
- Melodia技巧: 开启
```

### 不同乐器的推荐设置

#### 人声
```
- 音符起始阈值: 0.45
- 帧阈值: 0.25
- 最小音符长度: 150 ms
- 频率范围: 80-1000 Hz
```

#### 钢琴
```
- 音符起始阈值: 0.55
- 帧阈值: 0.35
- 最小音符长度: 100 ms
- 频率范围: 27.5-4186 Hz
```

#### 吉他
```
- 音符起始阈值: 0.50
- 帧阈值: 0.30
- 最小音符长度: 120 ms
- 频率范围: 82-1175 Hz
```

## 🔍 技术特点

### 使用的开源算法

1. **Basic Pitch (Spotify)**
   - 轻量级音频转MIDI模型
   - 支持多音转录
   - 支持音高弯曲

   - 检测相同音高的相邻音符
   - 合并间隔小于50ms的音符
   - 有效解决分段问题

## 💡 使用技巧

### 获得最佳效果

1. **先分离音频** - 使用 htdemucs_6s 模型获得更纯净的音轨
2. **选择合适音轨** - 人声、钢琴、吉他等单一乐器效果最好
3. **调整参数** - 根据音频特点调整阈值
4. **多次尝试** - 不同参数可能产生不同效果

### 处理复杂音频

1. **混合音频** - 先分离，再转换单个音轨
2. **噪音较多** - 提高音符起始阈值
3. **快速音符** - 降低最小音符长度
4. **音高不准** - 调整频率范围

## 🔧 故障排除

### 问题：转换器不可用

**解决方案**:
```bash
pip install basic-pitch
```

### 问题：音符太多/太少

**解决方案**: 调整音符起始阈值
- 太多 → 提高到 0.6-0.7
- 太少 → 降低到 0.3-0.4

### 问题：音符长度不准

**解决方案**: 调整帧阈值
- 太长 → 提高到 0.4-0.5
- 太短 → 降低到 0.2-0.3

### 问题：仍有分段

**解决方案**:
1. 降低音符起始阈值到 0.35
2. 降低帧阈值到 0.20
3. 增加最小音符长度到 200ms
4. 开启 Melodia 技巧
